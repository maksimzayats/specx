# modern-python-template Base Is Mandatory

Every generated project must keep the modern-python-template Base: FastAPI delivery, Django ORM and admin, dependency injection, architecture guardrails, tests, linting, and typing. This keeps the template focused on its intended identity instead of becoming a general Python project generator; project creators can still remove capabilities around Celery, authentication endpoints, throttling, storage backends, observability, documentation publishing, extra Docker services, GitHub workflows, and example domains.
