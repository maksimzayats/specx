## `foundation.imports-use-scoped-packages`

Reject imports from the removed alpha-stage `specx.foundation` namespace.

**Status:** Enabled by default.

### Why this exists

The removed alpha namespace did not communicate architectural ownership. Scoped foundation imports make the layer explicit.

### Failing example

```python
from specx.foundation import BaseUseCase
```

### Passing example

```python
from specx.core.foundation.use_case import BaseUseCase
```

### Detection notes

Scans source and test imports for specx.foundation and all of its submodules.
