## `services.require-effect-specific-bases`

Require core services to choose pure, read, or effect service bases.

**Status:** Enabled by default.

### Why this exists

Pure, read, and effect bases state what collaborators and operations a service may own.

### Failing example

```python
class PricingService(BaseService):
    ...
```

### Passing example

```python
class PricingService(BasePureService):
    ...

class OrderLookupService(BaseReadService):
    ...
```

### Detection notes

Requires classes under core scope services packages to inherit BasePureService, BaseReadService, or BaseEffectService directly or through project ancestry.
